# “诸神黄昏”签到系统 / _Rangarok_ check in system

## How to use

First, edit the `build/data/stuDB.js` to suit your situation.  
Then, run these commands:

```bash
npm i
npm start
```

## How to develop

First, edit the `build/data/stuDB.js` to suit your situation.  
Then, run these commands:

```bash
npm i --include=dev
npm run dev
```

## How to build for production use

First, edit the `build/data/stuDB.js` to suit your situation.  
Then, run these commands:

```bash
npm run tsc
```
